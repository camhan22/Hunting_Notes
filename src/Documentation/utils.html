class utils.ModelTrainer(logger, name, req_mod_cb=None, load_data_cb=None, training_kwarg_cb=None, training_cb=None, post_train_cb=None)

Bases: object

Handles training of models in non-blocking manner

Parameters:

  • logger (logging.Logger) – The logger handle to use while model training

  • req_mod_cb (Callable) – Callback that runs to load in any required models to train the current one

  • load_data_cb (Callable) – Callback that runs to load in training data for the model

  • training_kwarg_cb (Callable) – Callback that runs to load any information that needs to be passed to the training function

  • training_cb (Callable) – Callback that runs to train the model

  • post_train_cb (Callable) – Callback that runs after the model has finished training. Usually to save the model to a directory
