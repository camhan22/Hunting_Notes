class sidebar.sidebar.Sidebar(root, min_rel_detect, max_rel_display, min_rel_display: float = 0, side: str = 'left', default_tab_height: float = 0.1, animate: bool = False)

Bases: object

Class to handle the functions of the sidebar

Parameters:

  • root (ttkbootstrap.Window, ttkbootstrap.Frame, tkinter.Tk, tkinter.Frame) – The root window the sidebar should be in

  • min_rel_detect (float (0-1)) – The minimum relative width of the root display the sidebar should activate at

  • max_rel_display (float (0-1)) – The maximum size relative to the root window that the sidebar should expand to

  • min_rel_display (float (0-1)) – The minimum size the sidebar should be relative to the root window (Default to 0 so it doesn’t show)

  • side (str (left or right)) – The side of the root window the sidebar should be on

  • default_tab_height (float (0-1)) – The relative size tabs should be if no unique size is given (Defaults to 0.1)

  • animate (bool) – Whether to animate the sidebar expansion and collapse (Defaults to False)
