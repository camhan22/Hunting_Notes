Class that other dialogs inherit to handle basic functions common to most dialogs

Parameters:

  • main_window – The window to show this dialog in front of

  • is_modal (bool) – Determines if the user is able to close the dialog without pushing one of the given buttons (Defaults to True)

  • no_okay_cancel (bool) – Whether or not to automatically include the okay/cancel buttons at the bottom (Defaults to True)
