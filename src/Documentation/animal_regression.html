animal_regression package

This package implements the OneClassSVM model from sklearn to determine whether the selected times, dates, and weather conditions will lead to seeing the specified species at that location

Note

The OneClassSVM model may not be a good model for this application. It would be better for the model to predict how many of the desired species you should see instead of a yes or no. This is something I am working on.

animal_regression.animal_finder module

class animal_regression.animal_finder.AnimalFinder(root, data_directory: str, db_name: str, weather_fields: list, property_center: tuple, species_classes: dict, desired_species: str = 'Deer', time_interval: int = 15, first_week_day: str = 'sunday', retrain: bool = False)

Bases: object

Takes in current data to predict how many deer you should see at each stand location

Parameters:

  • root (ttkbootstrap.Window, ttkbootstrap.Frame, tkinter.Tk, tkinter.Frame) – The main window that this belongs to

  • data_directory (str) – The directory to look for any data the finder might need

  • db_name (str) – The name of the database that is being used

  • weather_fields (list[str]) – The list of fields to use for the weather when using the model

  • property_center (tuple(lat, long)) – The center of the property. We assume the property is small enough the variation in weather is negligible

  • species_classes (list[str]) – The list of species we can choose to hunt

  • desired_species (str) – The species we want to hunt (Defaults to Deer)

  • time_interval (int (units of minutes)) – The interval of time between data points to use

  • first_week_day (str) – The day to consider as the first of the week. (Defaults to sunday)

  • retrain (bool) – Whether to force a model retrain or not (Defaults to False)

Note

The OneClassSVM model may not be the best model for this type of problem, especially as other variables are added.
