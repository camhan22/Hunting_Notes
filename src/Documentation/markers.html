markers module

This module helps hold the data associated with a map marker such as name, latitude, longitude, notes folder path, images folder path
class markers.Marker(map_widget, lat, long, name, marker_type, data_directory, database)

Bases: object

Class to hold the data for each marker and operate on it

Parameters:

  • map_widget (tkintermapview.map_widget) – The tkintermapview widget the marker should be attached to

  • lat (float) – The latittude to put the marker at

  • long (float) – The longitude to put the marker at

  • name (str) – The name of the marker

  • marker_type (str) – The type of marker it should be (Camera, Stand, Point of Interest)

  • data_directory (str) – The directory where the data for the marker should go

  • database (str) – The name of the database we are using (The property name)
